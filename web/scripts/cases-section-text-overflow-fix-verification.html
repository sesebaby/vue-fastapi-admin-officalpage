✅ 修复内容总结

  • 使用n-ellipsis组件：替换n-text，自动处理文字溢出和省略号显示
  • 优化文字尺寸：调整字体大小和行高，确保在有限空间内正确显示
  • 添加溢出处理：设置max-height、word-break、overflow-wrap等CSS属性
  • 改进卡片布局：使用flex布局，合理分配空间，防止内容溢出
  • 响应式优化：不同屏幕尺寸下的文字大小和显示策略
  • 移动端适配：小屏幕下隐藏描述文字，避免重叠和溢出

🎯 技术实现要点

  • Naive UI兼容：使用n-ellipsis组件替代原生div
  • CSS Grid布局：确保卡片在不同屏幕下正确排列
  • 文字截断：-webkit-line-clamp: 3 限制最多显示3行
  • 弹性布局：flex: 1 让描述区域自适应剩余空间
  • 溢出隐藏：overflow: hidden 确保内容不会超出边界
